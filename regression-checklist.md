# Sample Defect Reports

## Defect ID: DONEISH-001

**Title**
Timer stops unexpectedly when navigating away from the screen.

**Steps to Reproduce**
1. Start a micro-commit timer
2. Navigate to another screen
3. Return to the timer screen

**Expected Result**
Timer continues running.

**Actual Result**
Timer resets to starting value.

**Severity**
Medium

---

## Defect ID: DONEISH-002

**Title**
Momentum tracker does not update after task completion.

**Steps to Reproduce**
1. Complete timer session
2. Log task completion

**Expected Result**
Momentum tracker increases.

**Actual Result**
Momentum tracker remains unchanged.

**Severity**
High
